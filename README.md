# 🏡 Home Rental Price Prediction

Regressão linear múltipla para prever o valor do aluguel de um imóvel.

## 🎯 Objetivo
Estimar o preço de aluguel usando variáveis estruturais do imóvel.

## 📊 Dataset
- Tamanho (m²)
- Quartos
- Idade do imóvel

## 🧠 Técnicas Utilizadas
- EDA
- Matriz de correlação
- Regressão linear múltipla
- Avaliação com RMSE e R²

## 🚀 Resultados
O modelo apresentou boa capacidade de previsão, com variáveis estruturais influenciando diretamente o preço.

## 🛠️ Tecnologias
Python, Pandas, NumPy, Scikit-Learn, Seaborn

## ▶️ Como rodar
Clone → abra o notebook → execute
