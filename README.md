# home-rental-regression
Modelo de regressão linear múltipla para prever preço de aluguel com base em tamanho, quartos e idade do imóvel.
# Previsão de Preço de Aluguel 🏡

Este projeto aplica **regressão linear múltipla** para prever o valor do aluguel de um imóvel com base em:

- Tamanho (m²)
- Número de quartos
- Idade do imóvel

## 📊 Objetivo

Demonstrar como modelos de regressão podem ser usados para estimar preços de aluguel utilizando múltiplas variáveis.

## 📂 Arquivos

- `home-rental-regression.ipynb` — Notebook com:
  - Análise exploratória (EDA)
  - Matriz de correlação
  - Treinamento do modelo de Regressão Linear
  - Avaliação (RMSE e R²)
  - Comparação entre valores reais e preditos

## 🛠️ Tecnologias
